### 19) request:

```bash
-- Заказы с именами пользователей
SELECT
    o.id         AS order_id,
    u.username,
    o.status,
    o.total_price
FROM orders o
JOIN users u ON o.user_id = u.id
ORDER BY o.total_price DESC;
```

### response:

```bash
 order_id | username |  status   | total_price 
----------+----------+-----------+-------------

        5 | diana    | completed |     5700.00
        6 | eve      | created   |     2990.00
(6 rows)
```

### Объяснение: 

Делаем JOIN (объединение) таблицы orders (с псевдонимом o) с таблицей users (с псевдонимом u) по внешнему ключу user_id, который связывает заказ с конкретным пользователем. Выбираем id заказа, имя пользователя, статус и общую сумму. Сортируем результаты по total_price в порядке убывания (DESC), чтобы самые дорогие заказы были первыми.
